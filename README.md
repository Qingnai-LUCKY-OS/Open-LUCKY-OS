# Open-LUCKY-OS
  LUCKY OS（青运操作系统） 是一款面向信创与国密合规场景的 Linux 发行版，代号「XOS」。它以 Linux LTS（7.0.3）内核为基线，叠加自研补丁集，构建木亥混合内核；搭载自研弦和（Xianhe）桌面环境、风徐调度器、明盾安全体系与 lky-pkg 包管理器，内置 SM2/SM3/SM4 国密算法。支持 x86_64 / aarch64 / risc-v3 /ram 四种架构，提供 ISO、LiveUSB、WSL、云镜像、容器与 PXE 等多种部署形态。LUCKY OS 致力于为政企与个人用户提供安全、自主、优雅的中国🇨🇳操作系统体验。
  我们的操作系统始终坚持自主创新的研发精神（并非全栈自研），但是我们还是会收录一些在我看来及其优秀并且对我的操作系统项目有巨大用处的项目（虽然我们并不会经常联系被收录项目的负责人或者开发者，但是请见谅，我暂时没有那么多时间，您也可以自发向我们联系）；但是保证：我们不会刻意去抹除任何一位被我们收录的项目的署名（具体可以在“Included items.md”查看）
