# Open-LUCKY-OS
  LUCKY OS（青运操作系统） 是一款面向信创与国密合规场景的 Linux 发行版，代号「XOS」。它以 Linux 内核的7.0.3版本为基线，并且预计长期停留在这一版本（具体见https://www.kernel.org网站）叠加自研补丁集，构建木亥混合内核（自研微内核+Linux7.0.3的自主精简修改版（大约删除20万行以目前眼光看来已经完全没有任何使用价值的冗余代码））；搭载自研弦和（Xianhe）桌面环境、风徐调度器、明盾安全体系与 lky-pkg 包管理器，内置 SM2/SM3/SM4 国密算法。支持 x86_64 / Ram64 / Risc-V64 /ram 四种架构，提供 ISO、LiveUSB、WSL、云镜像、容器与 PXE 等多种部署形态。LUCKY OS 致力于为政企与个人用户提供安全、自主、优雅的中国🇨🇳操作系统体验。
  我们的操作系统始终坚持自主创新的研发精神（并非完全全栈自研，但是自研占大部分），但是我们还是会收录一些在我看来及其优秀并且对我的操作系统项目有巨大用处的项目（项目严格遵守所收录各组件的开源许可证要求，详细引用清单及许可证声明见 Included items.md；并且保证：我们不会刻意去抹除任何一位被我们收录的项目的署名）
